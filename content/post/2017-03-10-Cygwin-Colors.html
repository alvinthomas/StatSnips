I recently discovered that Mintty is a great way to make your Cygwin terminal prettier. Mintty made my conversation from Fedora to Windows 10 and Cygwin more palatable.

Like all Cygwin pacakges/add-ons you’ll have to install it using the Cygwin installer. You can find some quick instructions at HowToGeek.

While there are a number of themes out there, I really like this Base-16 inspired set. I’ve also done some other tweaks to make my Cygwin terminal behave like my Mac Terminal (such as transparency when not in focus).
